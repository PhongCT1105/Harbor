# Philosophy

## Purpose
Record the principles that should guide Harbor design and contribution decisions.

## Scope
This file should capture design values, boundaries, and tradeoffs without specifying internal mechanisms.

## Project Pillars
- MCP Server Management: prefer clear visibility, operator control, and compatibility with existing MCP servers.
- Tool Calling Accuracy: prefer explainable selection, reduced tool noise, validation before execution, and feedback from outcomes.

- Contributors
- Users
- Plugin developers
- Core maintainers
- AI coding agents
