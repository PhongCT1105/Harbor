# Routing

## Purpose
Explain routing as a core Harbor concept.

## Scope
This file should describe how Harbor directs requests or tool intents toward appropriate MCP server capabilities at a high level.

## Project Pillars
- MCP Server Management: routing should respect server availability, health, configuration, and operator controls.
- Tool Calling Accuracy: routing should narrow broad requests toward relevant tool candidates before execution.
