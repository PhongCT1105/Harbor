# Contributing

## Purpose
Explain how contributors can participate in Harbor development.

## Scope
This file should cover contribution workflows, issue expectations, proposal guidance, review process, and community norms.

## Project Pillars
- MCP Server Management: contributions should preserve Harbor's role as a clear control plane for MCP server ecosystems.
- Tool Calling Accuracy: contributions should improve or protect the reliability, explainability, and evaluation of tool-calling behavior.

- Contributors
- Plugin developers
- Core maintainers
- AI coding agents
