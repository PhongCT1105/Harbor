# Agent Context

## Purpose
Give future AI coding agents the project context needed to make directionally correct changes.

## Scope
This file should summarize Harbor's product framing, architectural boundaries, and documentation expectations without defining APIs, schemas, or algorithms.

## Project Pillars
- MCP Server Management: agents should preserve Harbor's role as the place to inspect, configure, monitor, and control MCP servers and tools.
- Tool Calling Accuracy: agents should orient work toward better tool filtering, ranking, validation, routing, outcome tracking, and explainability.

## Intended Audience
- AI coding agents
- Contributors
- Core maintainers
