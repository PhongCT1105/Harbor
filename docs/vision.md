# Vision

## Purpose
Explain Harbor's long-term direction as an open-source MCP control plane and tool-calling intelligence layer.

## Scope
This file should describe the project mission, target problems, long-term goals, and non-goals at a high level.

## Project Pillars
- MCP Server Management: Harbor should help individuals and teams understand and control the MCP servers and tools they rely on.
- Tool Calling Accuracy: Harbor should make tool use more relevant, reliable, observable, and scalable for LLM clients.
